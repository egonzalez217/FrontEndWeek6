Create a new Garage by using the Garage Name form and clicking the Create New Garage Button.

From there the Garage can be removed, or cars can be added to the Garage using the Make and Model forms, and then the Add New Car Button.

A new row will be displayed to show the new car added to the garage, and then the car can be removed from the garage by clicking the Remove Car Button, or the car can be changed by using the Make and Model forms at the top of the table again, this time clicking the Change Car Button.
